这里是**文档区**——用来沉淀成体系的嵌入式知识：教程、原理、速查表。

和[博客](/)的区别：

- **博客**：按时间倒序的单篇文章，记录踩坑与思考，读完即走。
- **文档区**（你现在所在）：左侧有目录树，像一本可以随时翻阅、不断扩充的手册。

两个区共享同一套渲染能力（代码高亮、数学公式、时序图、可交互组件），但视觉上刻意做了区分，方便你一眼分清自己在哪。

## 怎么扩充

在 `src/content/docs/docs/` 下新增 `.md` / `.mdx` 文件，再到 `astro.config.mjs` 的 `sidebar` 里登记一下，左侧目录就会出现新条目。

> 下一步看看 [渲染能力演示](/docs/showcase)，确认公式、代码等都正常。
